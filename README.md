# 📩 SMS Spam Classification

An end-to-end Machine Learning and Natural Language Processing web application for detecting spam messages using TF-IDF vectorization and the Multinomial Naive Bayes algorithm.

---

## 🚀 Live Demo

https://sms-spam-classification-79u4.onrender.com

---

## ✨ Features

* 📩 Classifies Email/SMS as Spam or Not Spam
* 📝 Text preprocessing using NLTK
* 🔠 Tokenization
* 🚫 Stopword Removal
* 🌱 Porter Stemming
* 📊 TF-IDF Vectorization
* 🤖 Multinomial Naive Bayes Classifier
* 📈 Prediction Confidence Score
* 📋 Message Statistics

  * Character Count
  * Word Count
  * Processed Token Count
* 🧹 Displays Preprocessed Text
* 🔤 Displays Generated Tokens
* 📥 Download Prediction Report
* 🕘 Session Prediction History
* 💡 Sample Messages for Testing
* 🎨 Modern Streamlit User Interface

---

## 🧠 Machine Learning Pipeline

Input Message

↓

Text Preprocessing

* Lowercase Conversion
* Tokenization
* Stopword Removal
* Stemming

↓

TF-IDF Vectorization

↓

Multinomial Naive Bayes

↓

Prediction (Spam / Not Spam)

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/sms-spam-classification.git
```

Move into the project folder

```bash
cd sms-spam-classification
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
